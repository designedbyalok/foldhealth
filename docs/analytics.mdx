The **Analytics** workspace (`#/analytics`) gives leadership and operations teams 15 dashboard views over your patient population. Every view supports period selection, practice filtering, and drill-down into the underlying cohort.

## When to use it

- Run a weekly leadership review with the **Executive** view.
- Investigate a quality measure that is trending down.
- Model the financial impact of a new program in the **ROI Simulator**.

## The 15 views

| View              | What it answers                                                          |
|-------------------|--------------------------------------------------------------------------|
| Executive         | High-level KPIs across financial, quality, and operational health.       |
| Population        | Cohort size, demographics, and growth trends.                            |
| Financial         | Revenue, cost of care, and PMPM by line of business.                     |
| Risk              | Risk-stratification distribution and high-risk member counts.            |
| Quality           | HEDIS, Stars, and custom quality-measure performance.                    |
| Utilization       | Inpatient, ED, and primary-care utilization rates.                       |
| Care Coordination | TCM follow-up rates, outreach attempts, and program enrollment.          |
| Network           | In-network vs. out-of-network leakage by specialty.                      |
| Shared Savings    | Performance against shared-savings benchmarks.                           |
| ROI Simulator     | Project the financial impact of program changes.                         |
| Tool Usage        | How your teams use Fold Health day to day.                               |
| Platform Ops      | Uptime, error rates, and integration health.                             |
| AI Analytics      | Agent call volume, completion rates, and AI-handled minutes.             |
| SDOH              | Social-determinant indicators across the population.                     |
| Action Rules      | Performance of automated action rules and outreach campaigns.            |

## Use the filters

Every view shares the same filter row at the top:

1. **Period** — pick a date range (last 7 days, last 30 days, last quarter, year to date, or custom).
2. **Practice** — restrict to a single practice or roll up across all practices.
3. **Drill-down** — select a chart segment or table row to open a filtered cohort.

## Switch views

Use the left sub-nav to jump between views. Each view persists its filter state when you switch back.
